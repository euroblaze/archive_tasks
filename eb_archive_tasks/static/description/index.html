Archive Multiple Tasks

This module allows you to archive multiple tasks at once!

Very easy to use:

☛Select the tasks in list view -->

☛Action --> Archive Tasks

This was not possible before

From the stock odoo instalation and modules, this feature was not there.

Make sure you are in the list view, in order to select multiple tasks.

Saves time and work.

Click on all tasks, and list view

Select multiple tasks

Click the action Archive Tasks

From the wizard you can add or even remove

One click on Archive Tasks and you are done!
